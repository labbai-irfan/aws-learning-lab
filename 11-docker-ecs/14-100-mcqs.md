# Module 14 — 100 Docker & ECS MCQs (with Answers)

> Covers Docker fundamentals (images, containers, volumes, networks, Compose) and AWS containers (ECS, Fargate, ECR, task definitions, services). Attempt first, then check the **Answer Key** at the bottom. Aim for 80%+.
