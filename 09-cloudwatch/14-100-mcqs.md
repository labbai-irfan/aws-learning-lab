**Selected explanations**
- **8/9:** EC2 can't see guest memory/disk-used → install the agent.
- **11:** EMF embeds metrics in a log line; no extra PutMetricData calls.
- **30:** Default retention is "never expire" — always set it (cost!).
- **34:** `defaultValue=0` makes the metric report 0 so alarms evaluate reliably.
- **57:** Critical health metrics → `treat-missing-data=breaching`.
- **59:** `recover` self-heals on system status check failure.
- **75:** EventBridge cron() is UTC; use Scheduler for time zones.
- **96/97:** Insights bills per GB scanned — narrow the window.
