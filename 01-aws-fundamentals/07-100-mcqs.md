# Module 7 — 100 Multiple Choice Questions (with Answers & Explanations)

## Section A — Cloud Computing Basics (Q1–15)

D) Require a 10-year contract

A) Multiple data centers in one building

A) Servers run faster than light
B) Ability to experiment and deploy quickly

**9.** Converting CapEx to OpEx primarily helps by:
A) Increasing upfront hardware spend

**10.** Which is an example of "stop guessing capacity"?

**12.** "Go global in minutes" is possible because AWS:

B) Has infrastructure in regions worldwide you can deploy to quickly

**13.** Which statement about economies of scale is true?
A) Each customer negotiates hardware prices alone
B) Aggregated usage of many customers lowers per-unit costs

**14.** Which workload benefits MOST from cloud elasticity?

A) Resources are measured in meters
